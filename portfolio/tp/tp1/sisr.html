Java est un langage de programmation multiplateforme orienté objet qui s'exécute sur des milliards d'appareils dans le monde entier.

PHP (PHP Hypertext Preprocessor) est un langage de scripts Open Source conçu pour le développement d'applications web dynamiques.

HTML est un langage de balisage utilisé pour créer des pages web, permettant notamment de structurer le contenu et d’ajouter des liens hypertextes.
